What is Phylo ?

Phylo is a Java (JDK 1.5) phylogenetic framework developed at the University of Trieste, released under CC BY-NC-ND 3.0 License. It fully supports the use of certain algorithms of the phylip package, a free collection of programs for inferring phylogenies. The main features of Phylo are the following one:

  • GNU/Linux , Windows and MacOSX support
  • integration of phylip tools for inferring phylogenies based on distance matrices
  • creation of phylogenetics experiments, with possibility of saving, deleting and manipulate them
  • support for generic character matrices, with generic alphabet assignment
  • interactive phylogenetic tree visualization
  • character matrix editor, for creating, editing, loading and storing new matrices
  • possibility to extend the framework adding new algorithms
